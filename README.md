 📊 Superstore Sales Dashboard (Excel)

## 📌 Project Overview
This project is an interactive sales dashboard created in Microsoft Excel. It helps to analyze sales performance across different regions, categories, and time periods.

## 🎯 Objectives
- Analyze total sales and average sales
- Track number of transactions
- Understand regional performance
- Identify top-selling categories

## 📊 Key Features
- KPIs: Total Sales, Average Sales, Transactions, Quantity
- Region-wise and Category-wise analysis
- Monthly sales trends
- Interactive filters (Region & Year)
- Charts and Pivot Tables

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## ▶️ How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Use filters to explore the dashboard

## 💡 Insights
- Sales vary across regions
- Some categories perform better
- Monthly trends show peak sales periods
